CORS Nedir?

CORS (Cross-Origin Resource Sharing), web tarayıcıları tarafından uygulanan bir güvenlik protokolüdür. Bir web sayfasının, farklı bir kaynakta (alan adı, protokol veya port) bulunan kaynaklara (API'lere) HTTP istekleri göndermesini kısıtlar.

Nasıl Çalışır?

tarayıcı ==> <sunucu/proxy> ==> <api>

Tarayıcı ile hedef API arasında aracı görevi görür. Tarayıcı kısıtlamaları sadece web sayfaları için geçerlidir. Proxy bir sunucu tarafı uygulama olduğu için bu kısıtlamalara tabi değildir.
